# speer-back-end-assessment
1. Copy the .env.example file and rename it to .env.
```
cp .env.example .env
```

2. Install the required npm packages.
```
npm i
```

3. Start the development server.
```
npm start
```

4. Run in Production
```
npm run build
```

```
npm run server
```
